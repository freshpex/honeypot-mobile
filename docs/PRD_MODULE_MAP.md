# PRD Module Map

Source reviewed: `../HoneyPot Customer Mobile App.docx`, Draft v1.0, June 2026.

## Mobile Modules

- `auth`: signup, login, OTP, biometrics, onboarding checklist.
- `dashboard`: adaptive home, subscription status card, quick actions, recommendations.
- `subscriptions`: plan catalog, subscribe, pause, resume, renew, auto-renewal, gift plan.
- `meals`: weekly calendar, daily menu, selection, skip, nutrition tracker.
- `orders`: one-off menu, cart drawer, checkout, combo meals.
- `payments`: payment methods, payment confirmation, receipts, disputes.
- `wallet`: wallet top-up, auto-reload, transaction history, split payment.
- `deliveries`: live tracking map, status card, rider contact, delivery rating.
- `notifications`: preferences, notification center, WhatsApp opt-in.
- `profile`: personal info, addresses, dietary preferences, allergies, health goals.
- `referrals`: referral code, sharing, stats, referral history.
- `loyalty`: HoneyPoints, tiers, birthday offers, leaderboard entry.
- `support`: chat, FAQ, order-specific help.
- `community`: meal reviews, sharing graphics, leaderboard.
- `personalization`: recommendation explanations and preference-aware meal ordering.
